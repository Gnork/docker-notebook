# docker-notebook

1. Install `git`, `docker` and `docker-compose`.
2. Clone repository:

```bash
git clone https://github.com/Gnork/docker-notebook.git
cd docker-notebook
```

3. Create notebooks directory:

```bash
mkdir ~/notebooks
```

4. Start the notebook server:

```bash
docker-compose up notebook
```

5. Visit `http://localhost:8888`.
